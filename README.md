# 親なき後・知識引き継ぎテンプレート（Obsidian）

知的障害のある方の **「親なき後」** に備えて、家族や支援者が持っている **"言葉になっていない知識"（暗黙知）を記録し、次の支援者へ引き継ぐ**ための Obsidian テンプレートです。

専門知識もプログラミングも要りません。**メモがとれれば使えます。**

---

## これは何？

親や身近な支援者は、本人について膨大なことを知っています。

> 「大きな音が苦手」「青いタオルがあると安心する」「お風呂を無理に勧めると逆効果」「あの職員さんとは相性が良かった」

こうした知識は、医療カルテにも支援計画書にも書かれません。でも、**これが失われた瞬間、本人の生活は大きく揺らぎます。**

このテンプレートは、その知識を「本人を知らない次の支援者でも読んで分かる形」で残すための器です。お金を残す代わりに、**本人の人格と日常のプロトコルを残す**——そのための仕組みです。

## だれのため？

- 知的障害・発達障害のある方の **ご家族**
- **相談支援専門員・サービス管理責任者・グループホーム職員・生活支援員** などの福祉関係者
- 本人を新しく担当することになった支援者

## できること

- 本人の **人格・好き嫌い・こだわり** を一枚にまとめる（Person）
- 本人が **喜ぶこと／つらいこと** を記録する（Trigger）
- **試したこと・うまくいったこと・失敗したこと** を残す（Trial）
- 朝・食事・入浴などの **関わり方の手順** を残す（Protocol）
- 一覧表（**Bases**）で、本人・記録・要確認を自動で表示
- **公的制度・概念のリファレンス**（障害年金・成年後見・意思決定支援 ほか）を最初から同梱

## 使い方（3ステップ）

1. **Obsidian を入手** — [https://obsidian.md](https://obsidian.md)（無料・Win/Mac/スマホ）
2. **このフォルダを Obsidian で開く**（「フォルダを Vault として開く」）
3. **`はじめに.md` を開く** → 案内に沿って、本人を1人記録してみる

> くわしい手順は `つかいかた/` の各ページに。書き方の見本は `記入例/`（架空の本人「あおいさん」）にあります。

> 👥 複数の本人を担当する**計画相談支援専門員**の方は、まず `つかいかた/計画相談専門員のためのガイド.md` をご覧ください（本人ごとの整理のしかた）。

## 中身

| フォルダ | 内容 |
|---|---|
| `purpose.md` | この仕組みの「魂」。なぜ・何を大切にするか（最初に読んでほしい） |
| `はじめに.md` | 非技術者のための入口 |
| `つかいかた/` | インストール・本人の追加・AI併用・プライバシーの扱い |
| `templates/` | 各ページの雛形（12種） |
| `記入例/` | 架空の本人「あおいさん」の記入例 ＋ 地域版の例 |
| `データベース/` | 一覧表（Obsidian Bases） |
| `リファレンス/` | 公的制度・概念のおまけ参考知識（全国共通・要点） |
| `wiki/` | あなたが本人の記録を書いていく場所（最初は空） |

## AIと一緒に使えます（おすすめ）

Claude や ChatGPT などのAIに記録づくりを手伝ってもらえます。支援者が話すだけで、AIが適切なページの形に整えます。

> ⚠️ ただし、**本人の実名や、性・触法などの機微な情報をそのままクラウドAIに送ることには注意**が必要です。必ず `つかいかた/AIと一緒に使う場合.md` と `つかいかた/プライバシーの扱い方.md` を読んでください。

## リファレンス（参考知識）の更新について

`リファレンス/` の公的制度・概念は **全国共通の枠組み**をまとめたものです。制度は改正されます。各ページの「確認日」と公式リンクで最新をご確認ください。配布元で更新版が出たら、**`リファレンス/` フォルダだけを再ダウンロード**すれば、あなたが記録した本人データ（`wiki/`）はそのままに最新化できます。変更点は `リファレンス/更新履歴.md` を参照。

## プライバシーと尊厳について

このテンプレートは、本人の尊厳と **双方向の決定権**（本人の自己決定権と、本人が他者の決定権を尊重できる主体として育つこと）を土台にしています（→ `purpose.md`）。本名・連絡先などは手元の `raw/` に留め、記録では仮名・番号で扱うなど、`つかいかた/プライバシーの扱い方.md` の指針に従ってください。配布・共有は、本人（または成年後見人）の同意の範囲で行ってください。
