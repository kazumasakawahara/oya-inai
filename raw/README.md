# raw/ — 元データの保管場所（外に出さない）

このフォルダは、本人に関する元データや参考資料を置く、**あなたの手元だけ**の保管場所です。ここから必要な要点を `wiki/` に書き起こします（＝蒸留）。

## ⚠️ いちばん大事な約束

- **`raw/` の中身は、公開リポジトリやインターネット上のAIにそのまま出さない**でください。
  このテンプレートの `.gitignore` は、`raw/` の中身を誤ってコミットしないよう設定済みです（配布されるのは各フォルダの説明＝README だけ）。
- 記録（`wiki/`）の中では、本名ではなく**仮名や番号（P_001 など）**で本人を呼びます。
- 誰かに記録を渡すときは、原則 `raw/` を**含めず**、本人（または成年後見人）の同意の範囲で行ってください。

## 取り込み元（動線）ごとにフォルダを分けています

情報の「入ってき方」で分けると、**扱いの注意も一緒に分けられます。**

| フォルダ | 入れるもの | 扱いの注意 |
|---|---|---|
| `聴き取り/` | 親・本人・支援者の語り、面談・モニタリングのメモ・録音 | 機微・本人特定。外に出さない |
| `WEB資料/` | WEBから集めた参考記事（Web Clipper 等） | 他者の著作物。共有は自分の要約のみ |
| `AIチャット/` | LLM（Claude/ChatGPT 等）とのやりとり・下書き | 事実確認が必要。機微を送った記録に注意 |
| `書類・写真/` | 診断書・受給者証・契約・写真などの原本 | 機微・非公開 |

各フォルダの中は、必要に応じてさらにテーマ別に分けてもかまいません。各フォルダの README に、扱いの注意を書いています。

くわしい扱いは [[プライバシーの扱い方]] を読んでください。
