# raw/ — 元データの保管場所（外に出さない）

このフォルダは、**本人に関する元データ・機微情報**を置く、あなたの手元だけの保管場所です。

ここに入れるもの（例）:
- 本人の**本名・住所・電話番号**がわかる書類
- **診断書・処方・医療情報**の原本
- **受給者証・契約書・成年後見**関連の書類
- 面談の録音・写真・動画 など

## 約束

1. **`raw/` の中身は、公開リポジトリやインターネット上のAIに出さないでください。**
   （このテンプレートの `.gitignore` は、`raw/` の中身を誤ってコミットしないよう設定済みです。）
2. 記録（`wiki/`）の中では、本名ではなく**仮名や番号（P_001 など）**で本人を呼びます。
3. 誰かに記録を渡すときは、`raw/` を**含めず**、本人（または成年後見人）の同意の範囲で行ってください。

くわしくは [[つかいかた/プライバシーの扱い方]] を読んでください。
