# AIと一緒に使う場合

この仕組みは、**AI（Claude や ChatGPT など）に記録づくりを手伝ってもらう**ことを想定して作られています。忙しい現場でも続けやすくなります。ただし、**機微な情報の扱いには重要な注意があります**（このページの後半は必ず読んでください）。

## AIで何ができる？

- 支援者が「今日こんなことがあって…」と話すと、AIが**適切なページの形に整えてくれる**
- どの雛形（person / trigger / trial …）に当たるかを判断してくれる
- ページ同士のつながり（リンク）を提案してくれる
- 文章を、本人の尊厳を守る言い回しに整えてくれる

## 使い方の流れ

1. AIに、この Vault の中の **[[CLAUDE]]**（AI向けの指示書）と **[[purpose]]** を読ませる。
   - これでAIは「尊厳を守る」「失敗も書く」「本人を加害者扱いしない」といった原則を理解します。
2. その日の出来事を、話し言葉でAIに伝える。
3. AIが下書きを作る。
4. **あなたが内容を確認し、問題なければ保存する。**

> 🤖 どのAIでも使えますが、この Vault のファイルを直接読み書きできるAI（例：Claude Desktop、ファイル連携機能のあるもの）だと、よりスムーズです。

## ⚠️ いちばん大切な注意：何をAIに送ってよいか

多くのAIは、入力した内容を**インターネット上のサーバーに送って処理**します。本人の機微な情報を、そのまま送ることには慎重になってください。

### 送る前に立ち止まるべき情報

- **本人の実名・住所・電話番号・顔写真**
- **性に関すること**
- **触法（法律に触れる行為）に関すること**
- **詳しい医療情報・診断**

### どうすればよいか

1. **仮名にする。** 「たろうさん」ではなく「P_001」「本人」と書いてから渡す。
2. **機微な部分は要約・抽象化する。** 生々しい詳細はAIに送らず、自分で `wiki/sensitive/` に書く。
3. **特に慎重な情報は、ネットに送らないAIを検討する。** パソコンの中だけで動くAI（ローカルLLM。例：Ollama など）を使えば、情報が外に出ません。
4. 迷ったら送らない。**[[プライバシーの扱い方]]** を確認する。

## AIは「下書き役」。決めるのは人間

- AIの文章をそのまま信じない。**事実とちがう部分がないか、必ず人間が確認する。**
- 医療・法律・危機対応など重要な判断を、AIだけで決めない。専門職（相談支援専門員・医師・弁護士）に相談する。
- 本人の尊厳を守れているか、最後に人間の目でチェックする。

> AIは、あなたの記録を速く・楽にする道具です。しかし、本人を一番よく知っているのも、本人を守る責任があるのも、あなたです。
