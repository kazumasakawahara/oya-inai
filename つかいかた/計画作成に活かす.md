# 計画作成に活かす（記録 → 義務書類への"翻訳"）

> **ひとことで**：この記録は**追加の仕事ではなく、サービス等利用計画・モニタリング・アセスメントの"素材庫"**です。書いた内容が、そのまま義務書類の各欄に化けます。

## 対応表：記録 → 計画の欄

| この記録 | 主に効く欄（例） |
|---|---|
| **引き継ぎ1枚**・緊急連絡 | 緊急時対応・健康／医療 |
| Person「人格の核」・**Joy Trigger** | 興味・関心・強み／本人の好み |
| **Distress Trigger**・Protocol | 合理的配慮・環境・支援方法 |
| **Trial（特に失敗の記録）** | これまでの経過・支援上の工夫と留意点 |
| 意思決定支援の素材（→ [[CD_意思決定支援]]） | 本人の意向・選好 |
| Ecomap | 社会資源・関係機関・インフォーマル支援 |

## AIに下書きを頼む（任意）

AIを使えるなら、記録から**計画の下書き**を作れます。例：

> 「P_002 のページ群を読んで、サービス等利用計画の『興味・関心・強み』欄の下書きを作って」
> 「P_002 の失敗 Trial から、モニタリングで触れるべき留意点を箇条書きにして」

> ⚠️ AIの出力は**素材・下書き**です。事実確認と最終的な文責は専門員にあります。機微情報の扱いは [[AIと一緒に使う場合]]・[[プライバシーの扱い方]] に従ってください。

## ポイント

- **二重入力にしない。** 日々の気づきを記録に書いておけば、計画・モニタリングのときに**集めて使うだけ**。
- **「失敗の記録」は、計画の「留意点・支援の工夫」を具体化する最良の素材**です（→ purpose.md §3-2）。
- 複数の本人を担当する場合の整理は [[計画相談専門員のためのガイド]] を参照。
