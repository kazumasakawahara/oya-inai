# リファレンス（参考知識）について

このフォルダは、親なき後の支援でよく使う**公的制度・概念の「きほん」**をまとめた、おまけの参考知識です。

## 特徴

- **全国共通の枠組みだけ**を、短く・平易にまとめています。
- 窓口名・電話番号・金額など、**地域や時期で変わる細部はあえて書いていません**。「お住まいの自治体で確認」と案内しています。
- 制度は改正されます。**必ず最新の公的情報で確認してください。**

## あなたの地域版を作るには

このリファレンスの全国共通の枠組みを土台に、**お住まいの地域の具体情報**（どこに・誰に・何番に連絡するか）を足していくと、実際に使える形になります。その完成イメージは [[0_この地域の例について|地域の例（北九州市）]] を参照してください。

> 💡 地域版や本人別のページを作るときは、`templates/` の雛形を使い、関連ページを `[[ ]]` でつないでください。

## 参考資料を自分で増やすには

WEB上の障がい特性・対処法などの記事を集めて、自分の参考知識を増やせます。ブラウザ拡張（Obsidian Web Clipper）で `raw/web-clips/` に集め、要点を `wiki/concepts/` に書き起こす方法は [[参考資料をWEBから取り込む]] を参照してください。
