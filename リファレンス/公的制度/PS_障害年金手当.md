# 障害基礎年金・各種手当・心身障害者扶養共済

> **ひとことで**：障害のある本人の生涯の現金収入の柱。年金＋手当＋扶養共済の組み合わせが標準パッケージ。
>
> 📅 **確認日 2026-06-14** ／ ⚠️ 金額・所得制限・要件は改正されます。利用時は年金事務所・日本年金機構サイトで最新を確認してください。

> 親なき後の経済基盤の中核。**障害基礎年金（特に20歳前傷病）+ 特別障害者手当 + 障害年金生活者支援給付金 + 心身障害者扶養共済**の組み合わせが、知的障害のある方の生活保障の標準パッケージ。
>
> 制度の解釈論点・訴訟動向・不服申立て実務は[[CD_障害年金の解釈論点]]を参照。学生無年金訴訟を契機とする立法的救済は[[PS_特別障害給付金]]を参照。

## 1. 障害基礎年金

### 受給要件

次の3つすべてを満たす必要がある。

1. **初診日要件**: 障害の原因となった病気・けがの初診日が、国民年金加入期間中、**20歳前**、または日本国内在住60歳以上65歳未満で年金制度未加入の期間にあること
2. **障害認定日要件**: 障害認定日（または20歳に達した日）に、障害等級表の**1級または2級**に該当
3. **保険料納付要件**: 一定の保険料納付済期間+免除期間（**20歳前年金制度未加入期間に初診日がある場合は不要**）

### 等級の認定基準

- **1級**: 他人の介助を受けなければ日常生活がほとんどできないほどの状態。活動範囲がベッド周辺に限られる方など
- **2級**: 必ずしも他人の助けを借りる必要はないが、日常生活は極めて困難で労働による収入を得ることができないほどの状態。活動範囲が病院内・家屋内に限られる方など

> [!note] 精神・知的障害における等級認定の実務
> 精神障害・知的障害については「精神の障害に係る等級判定ガイドライン」（平成28年9月施行）が併用される。日常生活能力の判定（7項目4段階）と日常生活能力の程度（5段階）の組合せで「目安」が示されるが、就労状況・援助状況等を踏まえた総合判断とされる。詳細・最近の運用問題は[[CD_障害年金の解釈論点]]参照。

### 20歳前傷病による障害基礎年金の特則 ⚠️

知的障害支援で**最も重要**。保険料納付要件を問わない代わりに以下の制限がある。

- **所得制限**: 前年所得が **4,794,000円超**は全額停止、**3,761,000円超**は1/2支給停止
  - 扶養親族1人につき38万円加算（老人控除対象配偶者・老人扶養親族は48万円、特定扶養親族・控除対象扶養親族（19歳未満）は63万円）
- **支給停止期間**: 10月から翌年9月まで（前年所得を毎年確認）
- **その他**: 恩給・労災年金等を受給時の調整、海外居住・矯正施設入所中の支給停止

### 年金額（令和8年4月分から）

| 等級 | 年額 |
|------|------|
| 1級 | 1,059,125円 + 子の加算額 |
| 2級 | 847,300円 + 子の加算額 |

子の加算額: 2人まで1人につき243,800円、3人目以降1人につき81,300円。

### 請求方法

- **障害認定日による請求**: 認定日翌月分から受給。遡及は時効により5年分が限度
- **事後重症による請求**: 後に症状悪化で該当した場合、請求日翌月から受給。**65歳の誕生日の前々日まで**に請求が必要
- **基準傷病による請求**: 複数傷病の併合により等級該当となった場合（事後重症との使い分けは[[CD_障害年金の解釈論点]]論点9参照）

## 1-bis. 障害厚生年金（参考）

初診日に厚生年金被保険者であった場合は障害厚生年金の対象となる。知的障害（先天性）は出生日が初診日とされるため通常は対象外だが、以下のケースで関連する。

- 知的障害を伴わない発達障害（ASD・ADHD等）で20歳以降の初診日が厚生年金被保険者期間中であった場合
- 統合失調症・うつ病等の精神疾患を厚生年金被保険者期間中に発症した場合
- 知的障害がありながら厚生年金被保険者期間中に独立した精神疾患を発症した場合（併存疾患の処理は争点。[[CD_障害年金の解釈論点]]論点1参照）

障害厚生年金には1級・2級・3級＋障害手当金（一時金）があり、3級は厚生年金独自の給付。1級・2級は障害基礎年金と併給され、配偶者加給年金がある。

## 2. 障害年金生活者支援給付金

### 対象・要件

すべて満たすこと。

1. 障害基礎年金（旧法・旧共済の障害年金で政令で定めるもの含む）を受給
2. 前年所得が「4,794,000円 + 扶養親族の数 × 38万円（老人扶養等は48万円、特定扶養等は63万円）」以下

※障害年金等の非課税収入は所得判定に含まない。

### 金額（月額）

| 等級 | 月額 |
|------|------|
| 1級 | 7,025円 |
| 2級 | 5,620円 |

## 3. 特別障害者手当

- **目的**: 精神または身体に著しく重度の障害があり、日常生活で常時特別の介護を要する特別障害者の福祉向上
- **支給要件**: 著しく重度の障害により**在宅の20歳以上**で常時特別の介護を要する者
- **支給月額（令和8年4月より）**: **30,450円**
- **支払時期**: 毎年2月・5月・8月・11月に前月分まで
- **所得制限**: 受給資格者本人・配偶者・扶養義務者の前年所得が一定額以上の場合は不支給

## 4. 特別児童扶養手当

- **目的**: 精神または身体に障害を有する児童の福祉増進
- **支給要件**: **20歳未満**で精神または身体に障害を有する児童を家庭で監護・養育している父母等
- **支給月額（令和8年4月より）**:
  - 1級: 58,450円
  - 2級: 38,930円
- **支払時期**: 毎年4月・8月・12月に前月分まで
- **所得制限**: 受給資格者・配偶者・扶養義務者の前年所得が一定額以上で不支給

## 5. 心身障害者扶養共済制度（しょうがい共済）

### 制度の目的

障害のある方を育てている保護者が毎月掛金を納めることで、**保護者が亡くなった時などに、障害のある方に対し一定額の年金を一生涯支給**する制度。

### 親なき後支援における中核的意義

- 保護者死亡時にも障害のある方に**生涯にわたり年金が支給**される唯一の任意制度
- 割安な掛金、税制上の優遇あり
- **親が判断能力を有するうちに加入手続きすることが必須**（任意後見契約や遺言と併せて検討）

## 親なき後支援の文脈での活用ポイント

### 経済基盤設計の標準パッケージ

```
障害基礎年金（20歳前傷病・1級または2級）
  + 障害年金生活者支援給付金（月5,620〜7,025円）
  + 特別障害者手当（月30,450円、20歳以上で在宅常時介護）
  + 心身障害者扶養共済（親が加入していれば親なき後の生涯年金）
```

### 注意点

- **就労継続支援B型工賃**は所得制限の対象になる可能性。年所得3,761,000円超で半額停止、4,794,000円超で全額停止。工賃水準を意識した管理が必要
- **特別障害者手当 vs 施設入所**: 特別障害者手当は**在宅**が条件。施設入所支援を利用すると不支給になる
- **20歳到達時の年金請求手続き**: 20歳前傷病の場合、20歳到達時に障害認定日請求の手続きが必要
- **更新（有期認定）の警戒**: 精神障害は原則有期認定（1〜5年）。更新時の支給停止・級落ちは精神障害で増加傾向（[[CD_障害年金の解釈論点]]論点8参照）。親なき後を見据え、不服申立期限の管理体制を計画相談支援専門員・成年後見人で共有することが必要
- **2024年度不支給急増問題**: 2024年度の精神障害不支給率が前年の約2倍に急増し、2025年9月以降に職権取消しが進行中。該当時期の不支給事案は再検討が必要（[[CD_障害年金の解釈論点]]論点5参照）
