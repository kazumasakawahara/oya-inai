# 性と権利擁護

> **ひとことで**：性は、本人を被害から守ることと、本人が他者を尊重する主体として育つことの**両方**が最も切実に問われる領域。[[purpose]] §3-0 の双方向性原則の中核。
>
> 📅 **確認日 2026-06-14** ／ 機微情報を扱います。記録時は [[プライバシーの扱い方]] を参照。

> 性領域の中核概念。[[purpose]] §3-0「双方向の決定権の尊重」が**最も切実に問われる領域**として、性は [[CD_食と権利擁護]] と並ぶ二大テーマ。包括的性教育の枠組みを起点に、知的障害・発達障害のある方の性に関する権利と支援を整理する。

## このページの目的

機微情報の記述原則に従い、本ページの目的を明示する。本ページは以下の**3つの目的すべて**のために記録する:

- **本人を被害から守るため**（第一の柱）
- **本人が他者を尊重できる主体として育つ過程に伴走するため**（第二の柱）
- **法的論点の整理のため**（不同意性交罪・青少年保護等）

## 1. 包括的性教育の枠組み

### 国際的潮流

世界的には**包括的性教育（Comprehensive Sexuality Education, CSE）**として、人権とジェンダー平等を基盤とした幅広い性教育の流れが広がっている。その中核的な指針が、ユネスコ（UNESCO）等による**「国際セクシュアリティ教育ガイダンス」**である。

#### ユネスコ・ガイダンスの8つのキーコンセプト（2018年改訂版）
1. **人間関係**: 家族、友情、恋愛、子育て
2. **価値観、人権、文化、セクシュアリティ**: 権利と価値観の理解
3. **ジェンダーの理解**: 社会的構築と平等
4. **暴力と安全確保**: 同意、暴力の防止、ICTの安全性
5. **健康とウェルビーイングのためのスキル**: 意思決定、コミュニケーション、拒否、交渉
6. **人間のからだと発達**: 性と生殖の解剖生理、思春期
7. **セクシュアリティと性的行動**: 性行動、喜び
8. **性と生殖に関する健康**: 妊娠・避妊、HIV/STI

支援者向け性教育ガイドは日本の知的障害・発達障害のある方向けに、この国際的枠組みを適用・翻案したものと位置づけられる。

### 日本での状況

日本では従来、性の話題が「ナイーブ（繊細）」とされ、特別支援学校の高等部などでも性交や避妊について十分に教える機会が少ない、と支援者向け性教育ガイドは指摘。

**正しい知識がなければ自分で自分を守ることが難しくなる**。性の知識は本人を守るための力。

## 2. 二柱構造

[[purpose]] §3-0 を性領域に適用すると次のようになる。

### 第一の柱: 本人の側（被害から守る視点）

- **プライベートゾーンの守り方** → [[C_プライベートゾーンと身体の権利]]
- 嫌なことを「嫌」と言える権利
- 性的虐待・搾取からの保護 → [[PC_性被害時の対応フロー]]
- 性の健康・避妊・性感染症の知識 → [[C_性教育と健康相談]]
- 性的主体としての権利（恋愛・性行為・結婚・子を持つこと）

### 第二の柱: 他者との関係の側（本人が他者を尊重する主体として育つ伴走）

- **同意の概念** → [[C_性的同意と相互尊重]]
- 相手の「NO」を受け入れる
- 断られた後しつこく追わない
- 対等な関係（束縛・監視しない）
- 相手のプライバシー尊重
- 公共の場でのマナー（公然わいせつの回避）

両柱は対立せず**一つの原則の二側面**。支援者向け性教育ガイドが繰り返す合言葉は:

> **「自分も相手も大切に」**

これがそのままこの仕組み の双方向性原則と一致する。

## 3. 知的障害・発達障害のある方の性に関する特性

支援者向け性教育ガイドが指摘する特性（**本人を加害リスクのある対象として分類するためではない**。本人の特性を理解し、適切な伴走を設計するため）:

### 被害を受けやすくなる傾向

- 相手の言うことを信じやすい
- 強く「NO」と断ることが苦手
- 危険な誘いに気づきにくい
- 自己肯定感の低さから我慢してしまう
- 周囲に相談できず抱え込む

→ 障害のある女性は健常女性より **2〜3倍**性暴力を受けやすいという調査結果。

### 関係の境界線の理解への伴走が必要

- 他人との距離感がわからずボディタッチをして誤解を招く可能性
- 相手の同意・拒否のサインを読み取る練習が必要
- 公共の場での性的振る舞いのルール理解への伴走

→ ただし、これは本人を「加害リスクのある対象」として分類することではない。**本人が他者と豊かな関係を築ける主体として育つ過程**に支援者が伴走する文脈で記述する。

## 4. 関連する法律と制度

性に関する権利擁護は以下の法的・制度的枠組みで支えられる:

| 制度 | 関連内容 | 詳細 |
|------|----------|------|
| 不同意性交罪（2023年改正刑法） | 同意のない性交等は犯罪。性交同意年齢が13歳→16歳に引き上げ | [[PS_性犯罪関連法令]] |
| 各都道府県の青少年保護育成条例 | 18歳未満との淫行禁止 | [[PS_性犯罪関連法令]] |
| 公然わいせつ罪 | 公衆の面前でのわいせつ行為禁止 | [[PS_性犯罪関連法令]] |
| 障害者虐待防止法 | 性的虐待を含む5類型の虐待防止と通報義務 | [[PS_障害者虐待防止法]] |
| 障害者差別解消法 | 合理的配慮の提供（意思表明困難時は家族・支援者からの表明も含む） | [[PS_障害者差別解消法]] |
| 障害者総合支援法 | 意思決定支援の枠組み | [[CD_意思決定支援]] |

### 2023年の重要動向

- **不同意性交罪の創設**: 「相手の同意のない性交等は犯罪」が法律に明記
- 厚労省・自治体に対する**知的障害者の結婚や子育てを支援するよう**の通知 → 「障害者も自分の人生を自分で決めていい」という意思決定支援の流れ

## 5. 運用の指針

### Trial 記録のサブドメイン

- 性に関する出来事は trial_subdomain `sexuality` または `decision-rights-learning` で記録
- sensitivity が sensitive 以上の場合は `wiki/sensitive/` にも併載
- **「本人が他者の決定権を学ぶ過程の試行錯誤」**は decision-rights-learning として模索を蓄積

### Protocol ドメイン

- `private-time`（マスターベーション等の場面）
- `others-rights-learning`（他者の決定権を理解する継続的プロトコル）

### Sensitive 領域での記述

- 性的主体としての権利（第一の柱）と、本人が他者を尊重する主体として育つ伴走（第二の柱）の**両方**で記述
- 興味本位の詳細記述は禁止
- 本人の前で読み上げて恥ずかしい記述は書かない
- 加害・被害の二項対立を避け、本人を「加害リスクのある存在」として分類しない（A-9）
- 大幅な更新は、本人をよく知る中心的な支援者・家族の確認なしに行わない

## 6. 「模索」の明示

本人が他者の決定権を理解する過程に**確立された方法論は存在しない**:

> 本人の認知特性・コミュニケーション様式・過去の経験に応じて、現場の支援者一人ひとりが試行錯誤の中で見つけていくしかない領域

支援者向け性教育ガイドは現時点での標準的な枠組みを提示するが、個別の本人にどのようにアプローチするかは現場の Trial 記録として模索を蓄積していく。本ページの記述も**仮説**であり、運用しながら更新される。

## 7. この記録を読む支援者へ

支援者向け性教育ガイドが繰り返し強調するのは:

> **「自分も相手も大切に」**

本人の尊厳を守る関わり方と、本人が他者の決定権を理解できるよう支援する関わり方の**両方**を意識してください。

- **本人の側**: プライベートゾーンを守る、嫌なことを「嫌」と言える、性的主体としての権利を否定しない、被害が起きた場合の支援要請経路を関係者で共有
- **他者との関係の側**: 同意の概念を視覚支援・SST・絵カードで伝える、断られる経験を安全に積める機会を設計、本人が自分自身の「嫌だ」を表現できるようになる過程に伴走

両柱は同時に育つ。「自分も大切」が育つことが「相手も大切」の基盤になる。
