# 食と権利擁護

> **ひとことで**：「食」は本人の意思尊重と健康・安全がぶつかる支援の最前線。[[purpose]] §3-0「双方向の決定権」が最も切実に問われる領域の一つ。
>
> 📅 **確認日 2026-06-14**

> 「食」は知的障害支援の最前線。**本人の意思尊重と健康維持・安全確保のジレンマ**が日々生じる場面。

## 1. なぜ「食」が権利擁護の中核か

食は **生命維持・健康・楽しみ・コミュニケーション・自己決定**が交差する場面。
- 本人にとって: 1日3回×365日、人生で最も繰り返される自己決定の機会
- 支援者にとって: 健康維持の責任と本人意思の尊重がぶつかる場面
- 制度にとって: 提供方法・環境・人員配置すべてに関わる中核業務

ここで起きるジレンマは、他の場面（入浴・通院・服薬等）にも転用可能な思考の起点となる。

## 2. ジレンマの構造

### 二項対立として見える構図

```
本人意思の尊重         健康維持・安全確保
（食べたい/食べたくない）  （肥満防止/低栄養防止/誤嚥防止）
        ↕
        どちらを優先するか?
```

### 構造を反転させる視点

現場の実践資料 が示す解決の方向性:

> 「食べさせなければならない」という思いが、実は利用者の利益ではなく **支援者側の職務遂行上の利益** になっていないか

支援者中心の発想（「食べさせるのが仕事」）から、**本人主体の発想**（「楽しい食事の時間を保障する」）への転換が、二項対立を脱する道。

## 3. 食を巡る現場場面（現場の実践資料 §2 の要約）

### 食事拒否

- 障害特性・気分の問題、歯科治療等の精神的問題、食器の色・形の違い、混ざっているメニューなど
- 皿をひっくり返す行為が必ずしも「食べたくない」サインではない
- 解: 時間をかけて観察、多職種連携、楽しい食事の時間の保障

### 偏食・特定食品へのこだわり

- チーズばかり、白い物のみ、温度・調味料のこだわりなど
- 解: 食材ごとの分け盛り、チョイスボード、市販ドリンクで栄養補完、無理せず楽しい給食

### 過食・盗食・異食

- PWS による異常な食欲亢進、多飲（水中毒）、異食、盗食
- 解: 環境調整（見えないところにポット）、水筒で適量可視化、目の前から食物を排除、代替方法の提供

### 食事介助の問題

- 早食い・丸呑み・誤嚥リスク
- 解: 区切りのある製氷皿、一口大おにぎり、パフェスプーン、形態調整（ソフト食・ペースト食）

### 服薬

- てんかん治療薬・糖尿病治療薬の拒否
- 食事に混ぜる是非（LRA基準・記録必須）
- 向精神薬の食欲増進副作用との戦い

### 選択肢の制限

- 同じメニュー強制、外食・好物制限、食器・食事環境の制限
- 解: ルールを1つに絞る、デザートメニュー表、好きな食器の許容

### 「食べさせない」「食べさせ過ぎる」が虐待になる場面

| 区分 | 該当行為 |
|------|---------|
| 虐待 | アレルギー・トラウマがあるものの強制、無理矢理口に入れる、怒鳴る、羽交い絞め、病気もないのに食べさせない、交換条件にする |
| ネグレクト | 健康上問題のあるものを欲しがるまま渡し続ける |
| 支援 | 「食べてみようか」と促す、合理的配慮を行う |

詳細: [[PS_障害者虐待防止法]]、[[C_合理的配慮]]

## 4. 意思決定支援としての食支援

詳細: [[CD_意思決定支援]] §3 食の場面での意思決定支援

### 自己決定の支援ツール

- チョイスボード（調味料の選択肢提示）
- PECS（絵カード交換式コミュニケーション）
- 視覚的なメニュー表

### 拒否のサインの読み取り

- 言語化困難な利用者の意思を読み取るための観察ポイント
- 安心して「食べたくない」と表現できる関係づくり
- だまし討ち禁止

## 5. 権利擁護を支える組織的仕組み

現場の実践資料 §5 より:

### 多職種連携と組織的虐待防止

支援員単独で抱え込まず、組織として「**みんなで考え、みんなで統一した支援をする**」システムが不可欠。

### グレーゾーン事例の継続検証

山口県知的障害者福祉協会のスタッフ研究会モデル:
- 支援員2名 + 管理者1名 + 弁護士1名 + 学識経験者1名
- 各立場から食事拒否・服薬・過食制止・更衣強制・他害対応・交換条件の事例を検証
- **「グレーから白に変える」**には継続検証が不可欠

### 多職種連携・栄養ケアマネジメント

- スクリーニング → アセスメント → 栄養ケア計画 → モニタリング
- 保健衛生会議（看護師・支援員・施設長・管理栄養士、第2火曜）
- 給食会議（管理栄養士・支援員・支援課長・看護師、第3木曜）
- 中・高リスク者は週1回程度のミニカンファレンス

### PDCAサイクルと支援手順書

- 自閉スペクトラム症の特性上、支援者間のズレが利用者を混乱させる
- チーム全体の統一ルール（支援手順書）
- アセスメント → PLAN → DO → CHECK → ACTION

## 6. 法的・倫理的論点（弁護士・学識経験者の見解）

現場の実践資料 §4 より:

### 弁護士の視点

- **LRA（最小制約の原則）**: 他にも選びうる方法がないか
- **「誰の利益のための行動か」**: 支援者の職務遂行上の利益になっていないか
- **「手間を惜しまない」**: 効率優先は虐待への入口
- **契約論点**: 重要事項説明書・個別支援計画書での同意は重要だが、**羽交い絞め等の物理的拘束は依然として虐待**

### 学識経験者の視点

- **「支援なのか強制なのか」**: アレルギー・トラウマの強要は虐待、健康のための促しは支援
- **特性に応じた合理的配慮**: 量の調整・代替食品・環境調整
- **医学モデルから社会モデルへ**: 「させる」「指導」はタブー視される
- **理論武装**: 普段から正しい支援を行っていれば説明できる

## 7. 強度行動障害支援との接続

[[C_強度行動障害]] および [[C_氷山モデル]] と直結。食の場面は強度行動障害のアセスメントと環境調整の最前線:

- 過食・偏食・異食・盗食・多飲・食事拒否は氷山モデルの**水面上の表出**
- 水面下の特性（感覚過敏・予測不能への不安・伝わらない経験の蓄積）への介入が解決の道
- 視覚的構造化・代替方法の提供・安心できる環境が中核

## 8. 親なき後支援の文脈での活用ポイント

### Trial 記録のテンプレート

食を巡る Trial は、daily-life または decision-rights-learning サブドメインで記録。以下の観点を含めると検証しやすい:

- **状況**: 何が起きていたか（時刻・場所・直前の出来事・本人の様子）
- **試したこと**: 具体的な対応（誰が、どのような言葉・身振り・道具を使ったか）
- **結果**: 事実ベースで（本人の反応、その後の経過）
- **学び（仮説）**: 次に同じ状況になったらどうするか
- **氷山モデル考察**: 水面下にあった特性 × 環境要因の仮説
- **LRA 検証**: より制約の少ない方法はなかったか
- **誰の利益**: 利用者の利益のための介入だったか

### Protocol への昇華

機能した食支援は Protocol（meal ドメイン）として固定化:
- いつ、どこで、誰が
- どのような声かけ・道具
- 避けるべき関わり方
- 機能したトリガー（環境・道具・タイミング）

### Sensitive 領域への注意

食事拒否の背景に**過去のトラウマ**がある場合、Sensitive 領域として記録（sensitive_domain: trauma）。本人の前で読み上げて恥ずかしい記述は書かない。

### 親が暗黙知として持ってきた食支援

親が長年にわたって築いてきた「これだと食べる」「この時間なら大丈夫」という関わり方は、最大の引き継ぎ資産。Person ページの「人格の核」「現在の生活」と Trial で言語化する。
