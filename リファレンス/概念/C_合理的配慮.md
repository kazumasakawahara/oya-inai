# 合理的配慮

> **ひとことで**：力による制止や強制ではなく、**環境や提供方法を工夫して本人の権利を守る**実践。「制限が虐待に転じない条件」の中核。
>
> 📅 **確認日 2026-06-14**

> 障害者差別解消法に基づく概念だが、この仕組み では「**力による制止や強制ではなく、環境や提供方法を工夫することで本人の権利を守る**」具体的実践として位置付ける。「制限が虐待に転じない条件」の中核でもある。

## 1. 概念の射程

### 法的定義（障害者差別解消法）

合理的配慮の法的定義は、[[PS_障害者差別解消法]] で詳述。要点:

> 障害のある方が日常生活や社会生活で受ける様々な制限をもたらす原因となる**社会的障壁**を取り除くために、障害のある方に対し、**個別の状況に応じて行われる必要かつ合理的な配慮**

権利条約第2条は次のように定義:
> 障害者が他の者との平等を基礎として全ての人権及び基本的自由を享有し、又は行使することを確保するための必要かつ適当な変更及び調整であって、特定の場合において必要とされるものであり、かつ、均衡を失した又は過度の負担を課さないもの

### 義務主体（2024年4月から）

[[PS_障害者差別解消法]] の改正により、令和6年4月1日から **民間事業者にも合理的配慮の提供が法的義務**化された。**GH・生活介護事業所等の社会福祉法人やNPO法人を含む**。

### この仕組み での整理

合理的配慮とは、本人の障害特性に応じて、**力による制止や一律の強制ではなく、環境や提供方法を工夫すること**で、本人がその場面に参加できるよう支える具体的実践。法的義務であると同時に、[[purpose]] §3-0 双方向性原則の実装でもある。

## 2. 食の場面での合理的配慮（具体例）

現場で実践されている合理的配慮の具体例。

### 量の調整

- ご飯を口いっぱいにかき込む利用者には**製氷皿のように区切りのある食器**で一口量をコントロール
- **一口サイズのおにぎり**にして職員が一つずつ提供
- ヘッドが小さく平らな**介助用スプーン（パフェスプーン等）**に変更
- 過食傾向の方で「2杯食べないと気が済まない」場合は **1杯の量を少なくして2杯で通常の1杯分**にする

### 代替食品の活用

- お腹いっぱい食べたい人には**カロリーの少ない物で代用**
- 偏食により不足する栄養素は**市販ドリンクで補う**
- 牛乳でお腹を下す人には**温かいカフェオレを自分で作る**よう提案し、不足しがちなカルシウム摂取に成功した事例（横浜市事例集）

### 温度と形態の調整

- 常温のお茶を甘く感じる利用者には**冷蔵庫で冷やして提供**
- 咀嚼が少ない利用者にはおかずを**刻んで小鉢で分ける**
- 嚥下機能に合わせた**ソフト食・ペースト食**への変更

### 環境の調整

- 混ざっているものが苦手な人には**食材を別々に盛り付け**
- にぎやかな環境で集中できない子どもには**静かな部屋で少人数で食べる**
- 多飲のこだわりがある方には**見えないところにポットを隠す**、**水筒で適量を可視化**

### 選択肢の提示（自己決定の支援）

- **チョイスボード**: 調味料の選択肢を提示
- **PECS（絵カード）**: 「おかわり」などを自発的に表現
- **メニュー表**: 指差しで選べる
- 詳細: [[CD_意思決定支援]]

## 3. 制限が虐待に転じない条件

合理的配慮の境界線:

### 「支援」と「制限・強制」の境界線

| 区分 | 該当行為 |
|------|---------|
| **支援（合理的配慮）** | 健康を考えて「食べてみようか」と促す。利用者の特性を考えて代替品を用いたり量を調整する |
| **虐待** | トラウマやアレルギーがあるものを無理に食べさせる。怒鳴る。羽交い絞めなど物理的力で制止する。「○○しないとごはんあげないよ」と交換条件 |
| **ネグレクト** | 本人が欲しがるからといって、健康上問題があるものをそのまま渡し続ける |

### 適切な手続き

- **医師・専門職との相談**: 過食制止や服薬時に専門職に相談し指示を仰ぐ
- **記録の徹底**: やむを得ない対応は理由と経過を記録
- **複数職種による継続的な検討**: グレーゾーンの支援を放置せず、組織的に検証

## 4. 弁護士・学識経験者の判断枠組み

弁護士・学識経験者の見解（『これって虐待？虐待でない？』山口県知的障害者福祉協会 等）より:

### LRA（最小制約の原則、Less Restrictive Alternative）

「他にも選びうる方法がないか」を考える。複数の選択肢を比較して**より制約の少ない方法**を選ぶ。合理的配慮の実装において、これを満たさない介入は**過剰な制約**として批判される。

### 「誰の利益のための行動か」

「これをやらなければいけない（食べさせなければならない）」という思いが、利用者の利益ではなく**支援者の職務遂行上の利益**になっていないか。合理的配慮は本人の利益のための工夫であり、支援者の都合で簡略化されるものではない。

### 「手間を惜しまない」

短時間で効率的に済ませようとする支援は、しばしば本人の意思を踏みにじる。合理的配慮には**時間と工夫が必要**であり、それを惜しんで力で済ませることは虐待への入口。

### 学識経験者: 医学モデルから社会モデルへ

「10年前は普通だったことが、社会や環境が変わってきた」「医学モデルから社会モデルへ」という枠組みの転換。利用者の側を「治療」「指導」する発想から、社会・環境の側を整える発想へ。

## 5. 制度的位置付け

### 障害者差別解消法

詳細は [[PS_障害者差別解消法]] 参照。

要点:
- **正式名称**: 障害を理由とする差別の解消の推進に関する法律（平成25年法律第65号）
- **施行**: 平成28年4月1日。改正法は令和6年4月1日施行
- **不当な差別的取扱いの禁止**: 行政・民間事業者ともに法的義務（当初から）
- **合理的配慮の提供**: 行政は当初から義務、**民間事業者も令和6年4月から法的義務**
- 「事業者」には**社会福祉法人・NPO法人**を含む（=GH や生活介護事業所も対象）
- **意思の表明が困難な場合**は、家族・介助者等が本人を補佐して行う意思の表明も法律上の「意思の表明」に含まれる
- **建設的対話**による相互理解が重要
- 紛争解決: 「つなぐ窓口」・障害者差別解消支援地域協議会・主務大臣による行政措置

### 障害者総合支援法

[[PS_障害福祉サービス体系]] のサービス全体が、合理的配慮の制度的実装の側面を持つ。特に:

- 同行援護: 視覚障害者への移動の合理的配慮
- 行動援護: 強度行動障害状態の方への外出・行動の合理的配慮
- 共同生活援助の日中サービス支援型: 重度の方への住まいの合理的配慮

## 6. 親なき後支援の文脈での活用ポイント

### Trial 記録の評価軸

Trial で「試したこと」が合理的配慮として適切だったかを検証する評価軸:

- [ ] LRA を満たすか（より制約の少ない方法はないか）
- [ ] 誰の利益のための介入か
- [ ] 本人の意思が尊重されたか
- [ ] 結果として本人のQOLは向上したか

### Protocol への反映

機能した合理的配慮は Protocol として固定化し、後の支援者が継承できるようにする。

### 親が暗黙知として持ってきた合理的配慮

親が長年にわたって築いてきた「こうすると落ち着く」「これだと食べる」という関わり方は、しばしば合理的配慮の優れた実践であり、親なき後支援の最重要資産。Person ページや Trial で言語化することがこの仕組みの中核的価値。
