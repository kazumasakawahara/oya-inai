# 性自認と性的指向の多様性

> **ひとことで**：性のあり方は体の性別だけではない。性自認・性的指向の多様性を理解することは、本人が自分らしく生きるためにも、他者を尊重するためにも重要。
>
> 📅 **確認日 2026-06-14**

> 人の性のあり方は体の違いだけではない。**ジェンダー・性自認・性的指向**の多様性を理解することは、本人が自分らしく生きるためにも、他者の多様性を尊重するためにも重要。

## このページの目的

- **本人が自分の性のあり方を理解し受け入れるため**
- **本人が他者の多様な性のあり方を尊重できるよう伴走するため**

## 1. ジェンダーと性自認

### ジェンダーとは

**ジェンダー**は、生まれ持った体の性別に対する社会的・文化的な役割や、自分の認識する性別などを指す。

### 性自認（ジェンダー・アイデンティティ）

生まれたときに医師から「男の子」「女の子」と体の特徴で分けられるが、自分が心の中で「自分は男だ」「自分は女だ」「どちらでもない」などと感じる気持ちを **性自認** という。

### 性自認のパターン

- **多くの人**: 生まれたときの体の性と自分の性自認が一致
- **トランスジェンダー**: 体の性別と心の性別が一致しない人。例: 生まれたときは体が男性だったけれど心は女性だと感じる
- **ノンバイナリー**: 男性・女性どちらにも当てはまらない性自認を持つ人

### 重要な原則

> 性自認は本人の心の問題なので、他の人がとやかく決めつけてはいけません。それぞれの人が**自分らしく**感じる性で生きる権利があります。

## 2. 性的指向（セクシュアル・オリエンテーション）

### 性的指向とは

> 「**誰を好きになるか（どんな性別の人に恋愛感情や性的な感情を抱くか）**」ということ

### 性的指向のパターン

| 用語 | 内容 |
|------|------|
| **異性愛** | 男性なら女性を、女性なら男性を好きになる |
| **レズビアン** | 女性が女性を恋愛や好きの対象とする |
| **ゲイ** | 男性が男性を好きになる |
| **バイセクシュアル** | 男性も女性もどちらも好きになる（両性愛者） |
| **アセクシュアル** | 誰に対しても性的欲求を抱かない（無性愛） |
| **アロマンティック** | 誰に対しても恋愛感情を抱かない |

### 重要な認識

> 異性愛の人は世の中に多いので「当たり前」に思われがちですが、**異性愛だけが恋愛の形ではありません**。

## 3. 多様性を尊重するための原則

### 性自認・性的指向は変えられない

> 自分の性自認や性的指向は、自分で自由に選んだり変えたりできるものではありません。例えば、ゲイやレズビアンの人が「明日から異性を好きになろう」と思って簡単になれるものではないのです。

それは**生まれつきの性質**であり、その人の大切な個性。

### 差別・からかいはNG

> 自分と違う性のあり方をしている人に対して「おかしい」と決めつけたり、からかったり差別したりしてはいけません。

### 「自分は周りと違うかも」と感じている本人へ

> もし皆さんの中に「自分は周りの人と少し違うかも…」と感じている人がいても、それは決して悪いことでも恥ずかしいことでもありません。色々な性のあり方があって当然です。

## 4. LGBTQ+ という枠組み

近年、日本でも **LGBTQ+** という用語で性的少数者への理解を深め、偏見や差別をなくそうという動きが進んでいる。

- **L** Lesbian（レズビアン）
- **G** Gay（ゲイ）
- **B** Bisexual（バイセクシュアル）
- **T** Transgender（トランスジェンダー）
- **Q** Queer / Questioning（クィア／クエスチョニング）
- **+** その他多様な性のあり方

## 5. 知的障害・発達障害のある方への配慮

### 性自認・性的指向の自覚と表明

知的障害・発達障害のある方も、性自認や性的指向に関する自覚と表明の権利を持つ。本人がそうした感覚を表現するのに時間がかかったり、言語化が難しい場合があるため、支援者は:

- 本人の表現を否定しない
- 「男の子（女の子）だから」という決めつけをしない
- 服装・髪型・遊びの好みなどに性別の枠を強要しない

### 二重の差別への注意

支援者向け性教育ガイドが指摘する点:

> LGBTQ（性的少数者）の人への差別もそうですが、**障害者の性に関する差別も二重に起きやすい**と言われます。

知的障害があり、かつLGBTQ+でもある方は、二重の偏見・差別を受けるリスクが高い。支援者はこのことを意識する。

### 本人を主体とした記述

性自認や性的指向は本人の内面に属する事柄。Trial や Person ページに記録する場合、**本人の表現を尊重し、観察者の言葉で決めつけない**こと。

## 6. 親なき後支援の文脈での活用ポイント

### Person ページの「人格の核」での扱い

本人の性自認や性的指向に関する情報は、Person ページの「人格の核」セクションで本人を立体的に描く核となる。ただし:

- 本人がまだ自覚していない場合や表現できていない場合は記述しない
- 本人の言葉で記述する
- sensitivity は本人の状態と本人の意思に応じて判断（基本は internal、現在進行中の混乱や葛藤がある場合は sensitive）

### 関係構築の場面で

恋愛・性的関係に関する Trial を記録する際、**異性愛を前提にしない**。本人の性的指向を踏まえた支援を行う。

## 7. 障害者差別解消法との接続

[[PS_障害者差別解消法]] が示す合理的配慮の文脈で、**性的少数者である知的障害者**への配慮も求められる。例えば:

- ジェンダーに配慮した呼称（本人の希望する呼称・代名詞）
- 本人が安心して性自認・性的指向を表明できる関係づくり
- パートナーの認定（同性パートナーを家族として扱う等）

## 8. この記録を読む支援者へ

- 多様性を「特別なこと」「珍しいこと」として扱わず、**自然なこととして受け入れる**姿勢
- 本人が「自分は他の人と違うかも」と感じたとき、安心して相談できる関係を作る
- 知的障害があり、かつLGBTQ+でもある方への二重の偏見に注意
- 異性愛を前提とした表現を避ける（「彼女ができたら」ではなく「好きな人ができたら」など）
