# 氷山モデル（行動の理解）

> **ひとことで**：目に見える行動は氷山の一角。水面下の「特性×環境」に目を向ける見方。「困った行動」を「困っている表出」と読み替える。
>
> 📅 **確認日 2026-06-14**

> 強度行動障害および「困難な行動」の理解で中核となるモデル。問題行動を **「水面上の現象」** とし、その下に蓄積された **「特性 × 環境要因」** を見ていくフレーム。「困った行動」を「困っている表出」として読み替える視座を提供する。

## 1. モデルの核心

```
                       《水面上》
                       問題行動
                  （叩く・自傷・異食・
                    過食・食事拒否等）
        ─────────────────────────
                    水面（観察される境界）
        ─────────────────────────
                       《水面下》
       ┌─────────────────┐
       │ 障害特性                          │
       │  - 感覚過敏/鈍麻                │
       │  - 認知の特性                    │
       │  - コミュニケーションの困難│
       └─────────────────┘
       ┌─────────────────┐
       │ 環境要因                          │
       │  - 情報や刺激の偏り「分からない」│
       │  - 独自の表現が「伝わらない」経験│
       │  - 人や環境への不信感蓄積  │
       │  - 物理環境のミスマッチ    │
       └─────────────────┘
```

## 2. モデルの含意

### (1) 問題行動は「結果」であって「原因」ではない

水面上の行動を消そうとする支援（叱る・押さえる・物理的に制止する）は、水面下の蓄積を理解しないままの介入であり、**結果として虐待に転じる**ことがある。

### (2) 「分からない」「伝わらない」が積み重なる

- 情報や刺激が偏り「分からない」ことが積み重なる
- 独自の表現が「伝わらない」経験
- これが人や環境に対する**嫌悪感・不信感**となり水面下で蓄積
- 一定量を超えるとパニック・自傷・他害として表出

### (3) 介入は「水面下」に向けて

- 環境調整（構造化）で「分かる」を増やす
- コミュニケーション支援(PECS・絵カード・チョイスボード)で「伝わる」を増やす
- 安心できる環境（担当者固定・リズム尊重）で不信感の蓄積を減らす

## 3. 食の場面での氷山モデル

[[CD_食と権利擁護]] より:

### 食事拒否を例に

```
《水面上》皿をひっくり返す・食べない・大声を出す
        ↓
《水面下》皿の色が違う（視覚過敏）
        混ざっているメニュー（食感過敏・予測不能）
        だまし討ちで隠された食材への不信
        食事の場の騒がしさ（聴覚過敏）
        歯痛・体調不良
```

### 介入

- 皿を本人が好むものに固定
- 食材を別々に盛り付け
- だまし討ちをしない（口に入れるものを視覚的に明示）
- 静かな環境・少人数での食事
- 体調変化のチェック

## 4. ABC分析との接続

自閉症支援の「問題行動の理解と対応」の枠組みより:

**ABC分析**:
- **A (Antecedent)**: 先行事象（きっかけ）
- **B (Behavior)**: 行動
- **C (Consequence)**: 結果事象

これによって行動の **機能（注目・要求・逃避・感覚）** を推定する。

```
A: 朝、急に予定が変更された
↓
B: 異食（紙片を口に入れる）
↓
C: 支援者が止めに来る（注目を得る）
```

→ 機能: 注目要求 / 不安への対処

→ 介入: 予定変更時の事前通知、要求を伝えやすい代替手段（絵カード等）の提供

## 5. 視覚的構造化

水面下の「分からない」を減らすための工夫:

- レイアウトの変更（視覚的に整理された環境）
- 絵カードで活動を予告
- 一日のスケジュールを写真で提示
- 食事時の食器・食事環境の固定化

## 6. 氷山モデルが回避するもの

### 「行動を直す」発想

水面上の行動だけ消そうとする介入は、本人の不信感を強め、別の形で表出を強化する。

### 「本人の意思」と「行動」を直結する誤解

「皿をひっくり返した = 食べたくない」という単純な解釈は誤り。気持ちを切り替えるための行動かもしれないし、皿の色が違うだけかもしれない。**水面下を見る**ことが必要。

## 7. 親なき後支援の文脈での活用

### Trial 記録の組み立て方

Trial 記録を書くとき、状況・試したこと・結果に加えて、「水面下に何があったかの仮説」を書くと、後から読む支援者が氷山モデルで行動を理解できる。

### Trigger ページとの接続

Joy Triggers / Distress Triggers は、氷山モデルの「水面下」を構成する要素を一つひとつカード化したもの。Trigger 単体で見るのではなく、複数の Trigger が重なって水面上の行動が出現するという理解で運用する。

### Person ページの「人格の核」との接続

Person ページの「人格の核」（性格・好み・嫌い・こだわり）も、氷山モデルの「水面下」を厚く描くための情報源。本人の歴史を支援者全員で共有することが、氷山モデルに基づく支援の基盤。

### 「急に見える」状態悪化の読み解き

「急に」現れたように見える状態悪化（動作の緩慢化・発語減少・閉じこもり・興味の喪失等）も、氷山モデルが警戒すべき典型例の一つ。

```
《水面上》動作の緩慢化・発語の減少・閉じこもり・興味の喪失
        ↓
《水面下》（長期的に蓄積されてきたもの）
        - 視覚的記憶として残った過去の不快体験
        - 共感力の高さゆえの他者の苦痛の取り込み
        - 几帳面さによる過剰適応の限界
        - まじめさゆえに「いやな出来事」を我慢し続けた累積
        - 環境変化（親の入院・支援員交代等）への適応コスト
        - 身体疾患（甲状腺機能低下症・睡眠時無呼吸等）の影響
```

#### 通常の氷山モデルとの相違点

- 「急に」見えるが、水面下は **数か月〜数年単位の長期蓄積** がある
- 水面下の構成要素に **身体疾患**（甲状腺機能・睡眠・聴覚・視覚等）が大きな比重を占めることがある
- 「行動上の問題」として水面上の現象を処理すると、不適切な投薬等で本人を更に傷つけることがある

#### 介入の優先順位

1. **身体疾患の医学的除外診断**（甲状腺機能・睡眠時無呼吸・聴覚・視覚等）
2. **環境要因の特定と調整**（直近6か月〜1年の変化を時系列で洗い出す）
3. **温かく落ち着いた人間関係の維持**（リハビリ的介入をしすぎない）
4. **必要に応じた精神科介入**

#### 親なき後支援との結節点

- **親の入院・支援員交代** が引き金になり得るため、移行計画における警戒事項として常に意識する
- 本人の「いつもの様子」のベースラインを Person ページ・Trigger ページに蓄積し、変化に早期に気づける体制を作る
- 「第3の場」（家・学校/職場以外の通える場所）の事前確保が予防的に機能する
