# この「地域の例」について（必ずお読みください）

このフォルダのページは、**北九州市を例にして書かれた「地域の記入例」**です。

## これは"あなたの地域"ではありません

窓口名・電話番号・URL・処理時間などは、**すべて北九州市の例**です。あなたの地域では内容が異なります。**そのまま使わず、お住まいの市区町村に合わせて書き換えてください。**

## 何のための例か

[[0_リファレンスについて]] の「制度のきほん」は全国共通の枠組みだけを説明しています。実際の支援では、それを**あなたの地域の具体情報（どこに・誰に・何番に連絡するか）に落とし込む**必要があります。このフォルダは、その「落とし込んだ完成形はこうなる」という見本です。

- 障害福祉サービス体系（北九州市の記入例）
- 重度心身障害者医療費助成（北九州市の記入例）← 自治体独自制度の例
- 基幹相談支援センター（北九州市の記入例）← 地域の事業所の例
- 親なき後の窓口経路（北九州市の記入例）← 地域の手続きフローの例
