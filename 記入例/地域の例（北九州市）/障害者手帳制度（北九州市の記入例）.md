# 障害者手帳制度

> 障害者手帳は、身体障害者手帳・療育手帳・精神障害者保健福祉手帳の3種を総称した一般的呼称。制度の根拠法はそれぞれ異なるが、いずれも障害者総合支援法の対象となり、各種支援策が講じられる。

## 制度の概要

### 療育手帳（知的障害）

- **根拠**: 児童相談所または知的障害者更生相談所での判定。各自治体が判定基準等の運用を定める
- **等級区分**: A1/A2/B1/B2 等（自治体により表記が異なる）
- **再判定**: 原則 5年ごと。障害の程度・年齢によっては不要な場合あり
- **判定時間**: 北九州市では3時間程度

### 精神障害者保健福祉手帳

- **根拠**: 精神保健福祉法
- **対象**: 統合失調症・気分障害・てんかん・発達障害・高次脳機能障害等
- **等級**: 1級〜3級（精神疾患の状態と能力障害の状態の両面から判断）
- **有効期限**: 交付日から **2年が経過する日の属する月の末日**
- **更新**: 2年ごとの更新申請が必要。**更新通知は行われない**
- **更新申請可能時期**: 有効期限の3ヶ月前から

### 身体障害者手帳

- **根拠**: 身体障害者福祉法（北九州市では北九州市長が交付）
- **対象**: 視覚障害・聴覚平衡機能障害・音声言語そしゃく機能障害・肢体不自由・心臓じん臓呼吸器ぼうこう直腸等の機能障害（**永続するもの**）
- **等級**: 1級〜6級（身体障害者障害程度等級表に基づく）
- **更新**: 原則として更新なし。障害の状態が軽減される変化が予想される場合は再認定あり

## 親なき後支援における意義

- 本人が利用できる**障害福祉サービス・税減免・公共料金割引・交通機関割引等**の入口となる基礎資格
- 障害種別が複数ある場合、**それぞれの手帳を取得**することで支援の幅が広がる（例: 療育手帳と精神保健福祉手帳の両持ち）
- 親なき後の支援者交代時、手帳の有効期限管理は引き継ぎ事項として最重要
- 精神保健福祉手帳は2年ごと更新が必要で**自動通知されない**ため、ケース管理で必ずトラッキング

## 申請の流れ（北九州市）

すべての手帳の申請窓口は **お住まいの区役所保健福祉課**（高齢者・障害者相談コーナー）。

### 療育手帳

1. 区役所保健福祉課で相談・判定予約
2. 判定機関で判定:
   - 18歳以上 → E_北九州市障害福祉センター
   - 18歳未満 → 子ども総合センター（ウェルとばた5階、判定予約: 093-881-5055）
3. 判定結果は自宅へ郵送
4. 該当の場合、区役所保健福祉課で手帳発行

### 精神障害者保健福祉手帳

1. 必要書類を区役所保健福祉課に提出
2. 審査（2〜3ヶ月）
3. 決定通知書が届いたら区役所で受け取り

**必要書類**: 障害者手帳申請書、医師の診断書（精神障害の初診日から6ヶ月以上経過した後、精神保健指定医が記載）または年金証書等の写し、写真（縦4cm×横3cm、上半身・無帽）。マイナンバーがわかるものと本人確認書類が必要。

### 身体障害者手帳

1. 区役所保健福祉課で診断書・意見書（様式）を受け取り
2. 身体障害者福祉法第15条に定める指定医師に診断書・意見書の作成を依頼
3. 交付申請書に診断書・意見書、本人写真を添付して区役所保健福祉課に申請
4. 区役所で障害程度を審査、該当認定で手帳交付

## 注意点・落とし穴

- **療育手帳の再判定タイミングは本人の状態評価の機会**: 5年ごとの再判定は、本人の生活状況・支援ニーズの変化を支援者全体で共有する貴重な機会。
- **精神保健福祉手帳は更新通知なし**: 有効期限切れに注意。3ヶ月前からの更新申請、審査に2〜3ヶ月。早めの手続きが必要。
- **自立支援医療（精神通院）との同時申請**: 精神保健福祉手帳の新規・更新申請時、診断書の所定欄が記載されていれば自立支援医療と同時申請可能（PS_自立支援医療 参照）。
- **身体障害者手帳のマイナンバー**: 申請時にマイナンバーがわかるものが必要。

## 関連 Entity

- E_北九州市障害福祉センター: 18歳以上の療育手帳判定、身体障害者更生相談所機能
- E_北九州市障害福祉行政組織: 区役所保健福祉課・障害福祉企画課・障害者支援課
- 子ども総合センター（18歳未満の療育手帳判定、ウェルとばた5階）
